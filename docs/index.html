Core Modules

Agents

Commuter and pacer agents with configurable behavioral models: utility maximization, softmax choice, epsilon-greedy, and satisficing.

Incentives

Carpool rewards, pacer driving incentives, departure shifts, and transit promotion with eligibility checks and budget constraints.

Optimization

Budget-constrained allocation algorithms: greedy (0.63 approx), dynamic programming (optimal), genetic, and online secretary.

Behavioral Models

Multinomial logit, mixed logit, prospect theory, and regret minimization models calibrated from Hytch rideshare data.

Optimization Algorithms

Algorithm Approximation Ratio Time Complexity

Online (Secretary) 1/e ≈ 0.37 O(n)

Quick Start

# Clone the repository
git clone https://github.com/LNshuti/ihute.git
cd ihute

# Create conda environment
conda env create -f environment.yaml
conda activate ihute

# Build dbt models
cd dbt && dbt build

# Run the dashboard
cd ../app && python app.py
